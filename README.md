## 功能特性

| 功能 | 说明 |
|------|------|
| 📖 **投资语录** | 内置 349 条精选语录，覆盖巴菲特、芒格、格雷厄姆、费雪、霍华德·马克斯、达利欧等 |
| 📊 **LC 实时行情** | 自动识别碳酸锂主力合约，实时抓取广期所现价，每 60 秒刷新 |
| 🗓️ **每日语录算法** | 固定种子打乱 + 日期滑窗，每天展示不同的 100 条，相邻两天零重叠 |
| ⏰ **交易时间感知** | 非交易时段（收盘后/节假日）自动隐藏行情，只显示语录 |
| 💾 **位置记忆** | 关闭时自动记录窗口位置，下次启动从同一位置出现 |

## 安装与使用

### 1. 环境要求

- macOS（依赖 macOS 透明窗口特性）
- Python 3.9+

### 2. 安装依赖

```bash
pip install akshare
```

> `tkinter` 是 Python 标准库，通常已内置。若提示缺失，可通过 `brew install python-tk` 安装。

### 3. 直接运行

```bash
python3 lc_cat.py
```

后台静默运行：

```bash
python3 lc_cat.py > /tmp/lc_cat.log 2>&1 &
```

### 4. 打包成 macOS App（推荐）

将 `桌面语录.applescript` 编译为 App，双击即可启动：

```bash
osacompile -o ~/Desktop/桌面语录.app 桌面语录.applescript
```

或直接使用项目内附带的 `桌面语录.app`（如有）。

## 常见问题

**Q: 行情不显示？**  
A: 请检查是否在交易时段（工作日 9:00–11:30 / 13:30–15:00），收盘后仅显示语录。

**Q: 提示 `akshare` 找不到？**  
A: 执行 `pip install akshare` 安装依赖。

**Q: 窗口消失了找不到？**  
A: 删除 `~/.lc_cat_pos` 文件，重新启动即可回到初始位置（屏幕右上角）。

**Q: macOS 提示"无法打开，因为无法验证开发者"？**  
A: 右键 → 打开，或在「系统偏好设置 → 安全性与隐私」中允许。
