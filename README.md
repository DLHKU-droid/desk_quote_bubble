## 这是个什么东西？

你有没有过这种痛苦：

> 行情软件开着，领导走过来，慌得像个没写完作业被老师抓到的小学生。

本工具完美解决这个问题。

它看起来是一个**桌面励志语录**小组件，随机显示巴菲特、芒格、格雷厄姆等投资大师的金句——

```
┌──────────────────────────────────────────┐
│ 「在别人恐惧时贪婪，在别人贪婪时恐惧。」  ×│
│                                           │
│                            —— 沃伦·巴菲特 │
│                                 LC 82400 │
└──────────────────────────────────────────┘
```

领导路过，看到的是：*哦，这个同事在读投资格言，很上进嘛。*

你看到的是：碳酸锂主力合约今天又跌了 800 块。

## 使用场景

**场景一：开放式工位，领导随时绕圈**

气泡半透明、字体小、颜色低调，停在屏幕角落里丝毫不显眼。就算领导凑过来，看到的也只是一句「耐心是投资者最大的美德」——谁能说这不是在认真工作呢？

**场景二：开会投屏，不想暴露持仓**

行情软件一旦投屏，立即社死。本工具收盘后自动隐藏价格，只留语录，开会投屏稳如老狗。

**场景三：想盯盘，又不想被自己的焦虑拖垮**

每 60 秒刷新一次价格，旁边配一句「市场保持非理性的时间，可以比你保持偿付能力的时间更长」—— 看完价格，顺便被宽慰了一下，效果拔群。

**场景四：忘记看盘**

窗口始终置顶，不用切换软件，低头写需求文档，余光就能瞟到价格。专注工作与实时盯盘，就这样在视网膜边缘和谐共存。

## 隐蔽性设计

| 设计 | 效果 |
|------|------|
| 暖米黄气泡 + 金色边框 | 像系统自带的便签，毫无炒股软件气质 |
| 透明度 0.78，圆角设计 | 融入桌面背景，不抢眼 |
| 价格藏在作者署名后面 | `—— 沃伦·巴菲特  82400`，乍一看只是作者名字 |
| 收盘后自动隐藏价格 | 下班时间一到，变回纯语录，不留证据 |
| 无标题栏、无 Dock 图标 | 任务栏干干净净，截图不会穿帮 |

## 安装与使用

### 1. 环境要求

- macOS（透明窗口是 macOS 独有特性，Windows 用户请另寻他路）
- Python 3.9+

### 2. 安装依赖

```bash
pip install akshare
```

> `tkinter` 是 Python 标准库，通常已内置。若提示缺失，可通过 `brew install python-tk` 安装。

### 3. 直接运行

```bash
python3 lc_cat.py
```

后台静默运行（推荐，连终端窗口都不留）：

```bash
python3 lc_cat.py > /tmp/lc_cat.log 2>&1 &
```

### 4. 打包成 macOS App（强烈推荐）

编译成双击可用的 App，放进启动台，真正做到无痕启动：

```bash
osacompile -o ~/Desktop/桌面语录.app 桌面语录.applescript
```

双击启动，若已在运行会弹通知提醒，不会开两个。

## 常见问题

**Q: 行情数字不显示？**  
A: 检查是否在交易时段（工作日 9:00–11:30 / 13:30–15:00）。收盘了就别盯了，回家好好睡觉。

**Q: 提示找不到 `akshare`？**  
A: `pip install akshare`，装完再跑。

**Q: 窗口不见了？**  
A: 删掉 `~/.lc_cat_pos`，重启会回到屏幕初始位置。可能是上次拖到了屏幕外面。

**Q: macOS 提示"无法验证开发者"？**  
A: 右键 → 打开，或在「系统设置 → 隐私与安全性」里点允许。苹果觉得你在干坏事，其实你只是在摸鱼。
